The vast majority of EU legislation is enacted by Statutory Instrument under section 2(2) of the European Communities Act 1972 (UK). It looks like there are at least 6500 of these on Legislation.gov.uk. These laws will need to be examined over the coming months, as the government considers whether to repeal, replace or amend them.

Doteveryone has built this website as an experiment to see if it is possible to crowd-source useful data about that body of legislation, and to see if we can use it to make the whole process more transparent.

For example: is it possible to build a list of all the regulations that relate to workplace rights? What are all the regulations that relate to privacy, the digital economy or data?

This data doesn't exist anywhere - so we need to try and build it.

We have put this together very quickly, so please let us know if you find any bugs.
